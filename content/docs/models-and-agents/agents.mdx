# 智能体入门

Agent 常被翻译成“智能体”或“代理”。

普通聊天机器人通常是你问一句，它答一句。Agent 更强调围绕一个目标，拆解任务、调用工具、读取资料、执行步骤，并根据结果继续推进。

## 一个简单区别

| 类型 | 更像什么 |
| --- | --- |
| Chatbot | 会聊天的问答窗口 |
| Agent | 带工具箱、会分步骤做事的助手 |

## Agent 常见组成

- 目标：要完成什么任务。
- 模型：负责理解、规划和生成。
- 工具：浏览器、搜索、数据库、代码执行、文档读写等。
- 记忆：保存上下文、偏好或任务状态。
- 反馈：根据执行结果修正下一步。

## 先别被带偏

Agent 不是自动等于可靠。它越能调用工具、执行动作，越需要边界、权限、日志和人工确认。

所以在真实工作里，Agent 应该被理解成“有执行能力的工作流”，而不是一个永远正确的自动人。
