# 大模型怎么工作

这页先建立一个粗框架，不追求数学细节。

## 1. 模型处理的是 Token

大语言模型不会直接“理解一整篇文章”，而是把文本拆成 Token，再根据上下文预测后续内容。

Token 可以粗略理解成文本的计算单位，它可能是一个字、一个词，也可能是词的一部分。

## 2. Attention 决定重点看哪里

Attention 可以理解成模型在处理上下文时，用来判断“哪些信息更重要”的机制。

这也是为什么长上下文、提示词结构、示例位置会影响回答质量。

## 3. 训练靠大量数据和误差修正

Backpropagation（反向传播）是训练神经网络时常见的方法：模型先输出结果，再根据错误把调整信号往回传，更新参数。

普通人不用记公式，只要知道：模型能力不是写规则写出来的，而是在大量数据和训练过程中学出来的。

## 4. RAG 是给模型外接资料

RAG 可以粗略理解为“先找资料，再让模型基于资料回答”。

它常用于企业知识库、文档问答、客服和检索增强场景，目标是减少模型凭空编答案。

## 5. Bias 会影响输出

Bias 是偏差。偏差可能来自训练数据、标注方式、产品策略，也可能来自用户输入本身。

所以 AI 输出不能只看“像不像真的”，还要看来源、证据和适用场景。
