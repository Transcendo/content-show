# 生成式与多模态

生成式 AI 不只生成文字。它也可以生成图片、视频、音频、三维场景，或者把文本、图像、语音和视频放在同一个任务里处理。

这一类词主要解释两个问题：机器如何“看懂”视觉内容，以及模型如何在不同模态之间建立联系。

## 三个入口

| 入口 | 关键词 | 解决什么 |
| --- | --- | --- |
| 视觉理解 | CV、CNN、CLIP | 让机器识别和理解图像、视频。 |
| 内容生成 | Diffusion Models、GAN | 让模型生成新的图片或其他内容。 |
| 跨模态能力 | Multimodal、Cross-modal generalization、NeRF | 让不同类型的数据互相配合。 |

## 质量判断要回到任务

图像好不好，不只看好不好看。视频是否稳定、图文是否一致、人物是否可控、版权是否清楚、结果是否能交付，都比单次效果图更重要。
