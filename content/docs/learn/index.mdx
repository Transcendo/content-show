# 开始学习

AI 知路的目标不是把 AI 讲得更神秘，而是帮普通读者建立一张稳定地图：哪些词是基础概念，哪些词是模型机制，哪些词是产品方法，哪些词是前沿和风险讨论。

你不需要先学完数学、论文和工程栈。先知道每个词属于哪一层，再决定自己要继续深入基础、应用、产品还是治理。

## 推荐顺序

1. [学习路线](/learn/learning-path)：先知道该按什么顺序看。
2. [AI 术语百科](/glossary)：把高频术语放进正确目录。
3. [AI 基础概念](/fundamentals)：理解 AI、ANI、AGI、ASI、AIGC、生成式 AI。
4. [机器学习基础](/machine-learning)：理解数据、监督学习、泛化和过拟合。
5. [模型与训练机制](/model-mechanisms)：理解神经网络、参数、损失、优化和推理。
6. [大模型与提示工程](/llm-prompting)：理解 LLM、Token、RAG、Embedding、微调和对齐训练。
7. [生成式与多模态](/generative-multimodal)：理解图像、视觉、多模态和跨模态能力。
8. [智能体、产品与公司](/agents-products)：理解 Agent 为什么不只是聊天框。
9. [前沿、安全与治理](/frontier)：理解偏差、幻觉、对齐、AGI 和治理问题。
10. [算力与基础设施](/infrastructure)：理解模型为什么贵、慢，以及怎么部署。
11. [资料与来源](/resources)：查看本站常用资料入口和核查规则。

## 内容原则

- 先讲人话，再补技术词。
- 区分事实、解释、争议和推测。
- 尽量标出信息来源，避免把营销话术当知识。
- 先建立判断框架，再进入细节。
- 页面宁可少，也不能靠空话撑目录。
