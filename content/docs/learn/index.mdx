# 开始学习

这里是 Content Show 的主入口。

本站暂时不追求“百科全书式完整”，而是先把普通人最容易遇到、最容易混淆、最值得先理解的 AI 知识整理成稳定路径。

## 推荐顺序

1. [学习路线](/docs/learn/learning-path)：先知道该按什么顺序看。
2. [名词速查](/docs/learn/glossary)：先把高频术语放进正确位置。
3. [核心概念](/docs/fundamentals)：理解 AI、ANI、AGI、ASI、AIGC 的边界。
4. [模型与智能体](/docs/models-and-agents)：理解大模型、提示词、RAG 和 Agent。
5. [生成式应用](/docs/creative-ai)：把 AI 放回真实任务里判断。

## 内容原则

- 先讲人话，再补技术词。
- 区分事实、解释和个人判断。
- 尽量标出信息来源，避免把营销话术当知识。
- 目录可以扩展，但入口要保持清楚。
