# AI 名词速查

这页先用用户提供的 WaytoAGI 飞书“80+ AI 名词解释 / 名词卡片速查”作为素材线索。飞书表格当前可见的首批术语包括：AGI、AI、AIGC、ANI、ASI、Accelerator、Agents、Alignment、Attention、Backpropagation、Bias、CLIP、CNN、CV、ChatGPT、Chatbot。

本站先不照搬原表，而是把这些词分配进更适合 Content Show 的学习目录。

## 一、基础能力词

| 词 | 放在哪个目录 | 普通人理解 |
| --- | --- | --- |
| AI | [核心概念](/docs/fundamentals) | 所有人工智能相关技术的总称，范围最大。 |
| ANI | [核心概念](/docs/fundamentals/ai-agi-ani) | 狭义人工智能，只擅长特定任务。 |
| AGI | [核心概念](/docs/fundamentals/ai-agi-ani) | 通用人工智能，强调能跨任务解决问题。 |
| ASI | [AI 深度卡片](/docs/ai-card/asi) | 人工超级智能，强调在很多重要脑力任务上超过人类。 |
| AIGC | [AI 深度卡片](/docs/ai-card/aigc) | 用 AI 生成文字、图片、视频、音频、代码等内容。 |

| 词 | 放在哪个目录 | 普通人理解 |

| Backpropagation | [模型与智能体](/docs/models-and-agents/how-models-work) | 训练模型时把错误往回传，用来调整参数。 |

| CLIP | [生成式应用](/docs/creative-ai/media) | 把文字和图像联系起来的模型能力，常出现在图像生成和检索里。 |
