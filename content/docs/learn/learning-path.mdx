# AI 学习路线

这条路线参考了 WaytoAGI 公开知识库里“记忆、理解、应用、分析、评价、创造”的学习层次，也结合本站“普通人先能看懂”的目标重新压缩。

## 1. 先记住高频词

从 [名词速查](/docs/learn/glossary) 开始，先分清 AI、ANI、AGI、ASI、AIGC、Agent、Alignment、Attention、Bias 等词。

这个阶段不需要追求公式和论文，只要能听懂别人讨论的对象是什么。

## 2. 再理解核心概念

进入 [核心概念](/docs/fundamentals)，重点看三组问题：

- AI、ANI、AGI、ASI 的能力范围有什么不同？
- AIGC 和生成式 AI 为什么会突然变成大众工具？
- 当前 AI 是“已经实现的能力”，还是“对未来能力的预测”？

## 3. 看懂模型与智能体

进入 [模型与智能体](/docs/models-and-agents)。

普通人不需要先学完整机器学习课程，但应该知道：

- 大模型是在输入、预测和生成之间工作。
- Token、Attention、Embedding、RAG 是理解产品能力的关键词。
- Agent 不是“更会聊天”，而是让模型围绕目标调用工具、拆解任务。

## 4. 放进真实应用场景

进入 [生成式应用](/docs/creative-ai)。

按真实任务来理解 AI，比按工具名理解更稳定：

- 写作、总结、翻译
- 画图、视频、音乐、配音
- 编程、数据处理、自动化
- 数字人、3D、工作流

## 5. 学会找资料

进入 [资料来源清单](/docs/resources/source-list)。

AI 信息变化快，单靠二手总结容易失真。后续每篇内容都应该尽量回到官方文档、论文、可信媒体、研究报告或长期维护的知识库。

## 6. 最后讨论影响

进入 [前沿与治理](/docs/frontier)。

等你能分清能力、应用和限制，再看对齐、安全、法规、硬件、商业化和就业影响，会更不容易被标题党带偏。
