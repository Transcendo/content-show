# 核心概念

这一组目录解决一个问题：当别人说“AI 很强”“AGI 要来了”“AIGC 改变生产力”时，他们到底在说什么？

## 推荐阅读

- [AI / AGI / ANI](/docs/fundamentals/ai-agi-ani)
- [AIGC](/docs/ai-card/aigc)
- [ASI](/docs/ai-card/asi)

## 先记住这张图

| 范围 | 关键词 | 当前状态 |
| --- | --- | --- |
| AI | 人工智能总称 | 已广泛应用 |
| ANI | 狭义人工智能 | 当前主流 AI 基本都属于这一类 |
| AGI | 通用人工智能 | 仍有争议，尚未有公开公认实现 |
| ASI | 人工超级智能 | 未来设想和治理讨论 |
| AIGC | AI 生成内容 | 已经进入大众工具和产业流程 |
