# 机器学习基础

机器学习的核心不是“机器突然懂了世界”，而是模型从数据里学习规律，再把这些规律用于新输入。

这一类词帮助你判断 AI 输出为什么会好、为什么会错，以及为什么同一个模型换到新场景后表现会下降。

## 三种常见学习方式

| 方式 | 普通人理解 | 常见问题 |
| --- | --- | --- |
| 监督学习 | 给题目和标准答案，让模型学映射关系。 | 标签质量会直接影响模型。 |
| 无监督学习 | 没有标准答案，让模型自己找结构。 | 结果需要解释和验证。 |
| 强化学习 | 在环境里行动，根据奖励学习策略。 | 奖励设计错误会诱导奇怪行为。 |

## 训练不等于会用

模型在训练集上表现好，只说明它熟悉训练数据。真正重要的是泛化：遇到没见过的新数据、新任务、新用户时，是否仍然可靠。

这就是为什么训练数据、验证集、正则化、过拟合和欠拟合这些词很重要。它们不是算法工程师的专属术语，而是判断 AI 是否可信的基础。
