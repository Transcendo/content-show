机器学习的核心不是“机器突然懂了世界”，而是模型从数据里学习规律，再把这些规律用于新输入。

如果你想真正看懂 AI，这一组词最好别散着记，而要按一条学习路线来理解：**先知道机器学习是什么，再分清深度学习和三种学习方式，最后回到泛化和过拟合，判断模型到底靠不靠谱。**

> [!info]
> **这页的作用不是堆术语，而是告诉你机器学习该从哪里开始、按什么顺序往下读。**
