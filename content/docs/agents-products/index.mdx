# 智能体、产品与公司

普通人接触 AI，往往先接触产品名：ChatGPT、GPT-4、OpenAI、DeepMind、TensorFlow。学习时要先把产品、模型、公司和技术框架分开。

## 先分清四种对象

| 对象 | 例子 | 先怎么理解 |
| --- | --- | --- |
| 产品体验 | ChatGPT、Chatbot | 用户直接使用的入口。 |
| 模型或能力 | GPT-4、Agents | 产品背后调用的能力或系统形态。 |
| 公司机构 | OpenAI、DeepMind | 提供研究、模型或产品的组织。 |
| 工具框架 | TensorFlow | 开发者构建模型时使用的工程工具。 |

## Agent 为什么单独放一类

Agent 不是“更会聊天”的窗口，而是围绕目标执行任务的系统。它可能调用搜索、浏览器、代码、数据库或文档工具，所以风险也比单纯聊天更高。

判断 Agent 是否可用，要看权限边界、执行日志、失败回退和人工确认，而不是只看它回答得像不像人。
