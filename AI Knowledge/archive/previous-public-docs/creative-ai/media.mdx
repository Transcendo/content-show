# 图像、视频与音乐

AIGC 不只生成文字，也生成图片、视频、音频、配音、数字人和 3D 素材。

## 先按输出类型分

| 类型 | 典型用途 | 关键风险 |
| --- | --- | --- |
| AI 图像 | 封面、海报、概念图、电商图 | 版权、人物一致性、细节错误 |
| AI 视频 | 短片、广告、演示、转场素材 | 稳定性、时序一致性、成本 |
| AI 音乐 | BGM、旋律草稿、音效 | 风格相似、版权、商用范围 |
| 数字人 | 口播、客服、培训、直播辅助 | 肖像授权、误导性、可信度 |
| CV | 图像识别、检测、分割、理解视频 | 数据偏差、误识别、隐私 |

## CLIP 为什么常被提到

CLIP 这类模型把文字和图像放到同一个理解空间里，让“用文字找图”“用文字指导生成图片”变得更可行。

普通人可以把它理解成：AI 开始能把“描述”和“画面”关联起来。
